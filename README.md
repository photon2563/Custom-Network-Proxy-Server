# Async Proxy Server

A production-style async HTTP/HTTPS forward proxy implemented in Python.

## Features
- Async event-driven architecture
- HTTP & HTTPS (CONNECT) support
- TLS SNI inspection
- Policy engine
- Metrics endpoint
- Hardened against abuse
- Layered YAML configuration
- Graceful shutdown
- CLI + script-based execution
